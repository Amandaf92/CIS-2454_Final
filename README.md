# CIS-2454_FinalT
Final project for CIS-2454.

## Project Description

This full stack shopping list app allows users to create shopping lists for different stores. Users can:

- Create new stores
- View all stores
- Delete stores
- Add items to selected stores
- Edit item name and quantity
- Mark items check/unchecked
- Delete items

## Features 

- GET requests
- POST requests
- PUT requests
- DELETE requests
- Dynamic frontend update using React state relational database structure
